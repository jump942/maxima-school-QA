5.Algorithm: Сварить пельмени
1) Взять кастрюлю;
2) Налить воды из крана;
Definition of Ready (DoR): В кране должна быть холодная вода 
Definition of Done (DoD): Холодная вода бежит
3) Поставить на плиту;
4) Включить плиту на максимальную температуру;
5) Достать пельмени из морозилки;
6) if ( вода закипела ) 
{
	кладем пельмени в кастрюлю;
}
else (вода не закипела) 
{ 
	do (вода не кипит)
	{ 
		ждать; 
	}
	while (вода кипит);
	{
		кладем пельмени в кастрюлю;
	}
}
7) Перенешиваем пельмени;
8) Ждем 5 мин;
9) Достаем тарелку;
10) Вылавливаем пельмени ложкой на тарелку;
Result: Пельмени готовы
